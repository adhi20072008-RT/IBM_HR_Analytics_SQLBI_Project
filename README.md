# IBM HR Analytics Dashboard

## Project Overview

This project focuses on analyzing employee attrition, workforce demographics, job satisfaction, overtime trends, and salary patterns using SQL and Power BI. The objective is to identify key factors influencing employee turnover and provide actionable insights through an interactive dashboard.

---

## Tools & Technologies

* MySQL
* Power BI
* DAX
* Microsoft Excel

---

## Dataset

IBM HR Analytics Employee Attrition Dataset

The dataset contains employee information such as age, department, job role, salary, overtime status, job satisfaction, marital status, years at company, and attrition details.

---

## Data Validation & Quality Checks

The following data validation steps were performed using SQL:

* Row Count Validation
* Null Value Analysis
* Duplicate Record Detection
* Data Profiling
* Data Consistency Verification

---

## SQL Analysis Performed

* Calculated Employee Attrition Rate
* Department-wise Attrition Analysis
* Job Role-wise Attrition Analysis
* Overtime Impact on Attrition
* Gender-wise Analysis
* Marital Status Analysis
* Job Satisfaction Analysis
* Salary Analysis
* Employee Retention Analysis
* Department Ranking Analysis using SQL Window Functions

---

## Key Insights

* Research & Development department experienced the highest employee attrition.
* Employees working overtime showed higher attrition compared to those not working overtime.
* Attrition was concentrated in specific job roles.
* Employee satisfaction levels appeared to influence retention patterns.
* Workforce demographics provided valuable insights into employee turnover trends.
